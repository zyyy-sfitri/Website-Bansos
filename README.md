Website Bansos

Website Bansos adalah aplikasi berbasis web yang digunakan untuk membantu proses pendataan, verifikasi, dan monitoring penyaluran bantuan sosial secara terkomputerisasi. Sistem ini dibuat untuk meminimalisir kesalahan data, mempercepat proses seleksi penerima bantuan, serta meningkatkan transparansi penyaluran bansos.

Website ini dikembangkan menggunakan framework CodeIgniter 4 dan dirancang agar mudah digunakan oleh admin maupun petugas lapangan.

Fitur

Website Bansos memiliki beberapa fitur utama, antara lain halaman login admin, pengelolaan data penerima bantuan, pengelolaan jenis bantuan, proses verifikasi kelayakan penerima, serta laporan data penerima bansos yang dapat digunakan sebagai bahan evaluasi.

Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan PHP dengan framework CodeIgniter 4, database MySQL, HTML, CSS, JavaScript, dan Bootstrap untuk tampilan antarmuka.

Cara Menjalankan Aplikasi

Pertama, clone repository ini ke komputer lokal.
Kemudian buat database baru di phpMyAdmin dan import file database yang tersedia.
Atur koneksi database pada file .env.
Setelah itu jalankan server lokal menggunakan perintah php spark serve.
Akses aplikasi melalui browser menggunakan alamat http://localhost:8080.

Tujuan Pembuatan

Website ini dibuat sebagai proyek perkuliahan untuk membantu proses digitalisasi sistem pendataan bantuan sosial dan sebagai media pembelajaran pengembangan aplikasi web berbasis framework.
